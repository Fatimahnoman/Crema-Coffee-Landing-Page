# Crema Coffee House — Landing Page

A single-file, responsive landing page for **Crema Coffee House**, a tiny corner coffee shop in the old town. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no libraries.

## Features

- **Single HTML file** — everything in one place
- **Fully responsive** — mobile, tablet, desktop (5 breakpoints)
- **Dark chocolate brown theme** with caramel accents
- **Hero section** with full-screen background image and dark overlay
- **Scroll-triggered animations** using Intersection Observer
- **Sticky navigation** with scroll spy and mobile hamburger menu
- **Menu grid** with image cards and prices
- **Image gallery** with hover effects
- **Customer review / quote section**
- **Visit info cards** (hours, location, phone)
- **WhatsApp CTA** with direct chat link
- **Keyboard accessible** with `:focus-visible` and skip-to-content link
- **`prefers-reduced-motion`** support

## Tech Stack

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (Intersection Observer, scroll spy)
- Google Fonts: Playfair Display + Inter
